## Meta Exchanger

Tools for Exchange Followers Facebook PRO

### Fitur:

- **Login by Cookies**: Cookies disimpan secara lokal di setiap perangkat, memastikan keamanan tanpa ada yang tersimpan di server.
- **Tambah Profil Pengguna**: Mudah tambahkan profil dengan menyertakan link dan label.
- **Hapus Profil Pengguna**: Hapus profil sesuai kebutuhan.
- **Dukungan Saat Ini**: Saat ini mendukung profil, halaman akan ditambahkan di versi 2 (segera hadir).

### Tutorial:

1. **Ekspor Cookies:**
   - **Chrome**: [Export Cookie JSON File](https://chromewebstore.google.com/detail/export-cookie-json-file-f/nmckokihipjgplolmcmjakknndddifde?hl=en)
   - **Firefox**: [Export JSON File for Puppeteer](https://addons.mozilla.org/en-US/firefox/addon/%E3%82%AF%E3%83%83%E3%82%AD%E3%83%BCjson%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB%E5%87%BA%E5%8A%9B-for-puppeteer/)

2. **Tambah Akun:**

  Setelah di install di chrome/firefox, export json nya lalu masuk ke menu Connected Accounts , paste semua cookies di kolom Cookies, lalu berinama Account Labelnya (INGAT, cookies di simpan di masing2 device kalian, tidak ada yang disave di server) 

3. **Share Akun di Publik:**

Setelah akun terpasang, jika ingin akun kamu difollow oleh orang2 masuk ke menu Synced Accounts, kasih nama Label FB di kolom Label Name (Ini nanti akan muncul di publik ketika orang2 akan follow di menu List) lalu isi kan link dengan url facebook kalian ( contoh : masukkan link nya seperti ini : https://www.facebook.com/TuZKi666/ ) klik add accounts

4. **Go Get Followers**

Setelah akun ditambah maka akun kalian akan muncul di orang2 (kecuali akun sendiri tidak akan muncul di list) jadi sistem nya kalian akan follow orang2 dan kalian juga akan di follow oleh orang2 (yang terdaftar sebagai workers)
